# Ganeshotsav IITG 2025
Full-stack event management platform for IIT Guwahati's annual Ganeshotsav festival.

## Tech Stack
- Frontend: HTML, CSS, Vanilla JS, Canvas API
- Backend: Node.js, Express.js, MongoDB
- Payments: Razorpay integration
- Email: Nodemailer (Gmail SMTP)
- Auth: JWT

## Features
- Razorpay payment gateway for merch orders and donations
- Automated email receipts via Nodemailer  
- JWT-protected admin dashboard with live stats
- Real-time transaction tracking
- Horizontal scroll gallery, blob reveal animations
- Mobile responsive design

## Setup
1. Clone the repo
2. cd backend && npm install
3. Create backend/.env (see .env.example)
4. npm run dev
